# note公開設定

## 基本設定

| 項目 | 設定値 |
|---|---|
| 価格 | **500円** |
| 形式 | 単体記事（マガジン未所属） |
| タグ | #note #AI副業 #ChatGPT #副業 #初心者 #プロンプト |
| カテゴリ | ビジネス > 副業 |

## 無料/有料の切れ目

### 採用位置
**第2章末「3回の失敗は、全部この3点のどこかが抜けていた、それだけの話でした。」の直後**

### 配分
- 無料: リード + 第1章 + 第2章（約2,500字 / 全体の約30%）
- 有料: 第3〜9章（約6,000字 / 全体の約70%）

### 理由
- 共感の山（読者が最も「自分のことだ」と感じる瞬間）で切れている
- "3点とは何か"が次章で明かされる構造＝続きを読まずにいられない
- 30%ラインはnote公式推奨ゾーンと一致

### 予告文（切れ目直前に配置済み）
```
ここから先は、実際に僕が1,500円売れるまでに使った
全プロンプト・無料/有料の切れ目の判定ルール・公開初日のX投稿テンプレ3本を、
一切伏せずに公開します。
```

## サムネイル

### 採用キャッチコピー（2行構成）
**メインコピー**: `1,500円稼いだ全手順`
**サブコピー**: `AIで500円noteを売った7ステップ（文章経験ゼロ）`

### デザイン指針
- 背景: 濃紺または濃い緑（信頼色）
- メインコピー: 白 or 黄で太字、フォントサイズ最大
- サブコピー: 白、メインの40%サイズ
- 右下または左下に「全プロンプト公開」のバッジ風装飾
- 人物写真/フリー素材は使わず文字だけで勝負（具体性重視）

### 代替キャッチコピー案（次回以降の差し替え用）
1. 1,500円稼いだ全手順｜AIで500円noteを売った7ステップ ← 採用
2. 3回失敗した会社員が、4本目で初売上
3. 文章経験ゼロ・フォロワー80人でも売れたnote作成術
4. コピペOK｜AIで売れるnoteを作る全プロンプト公開
5. 【保存版】AIに丸投げをやめたら売れた
