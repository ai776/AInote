# 【1,500円稼いだ全手順】ChatGPTに3回裏切られた会社員が、AIで500円noteを売るまでに作った7ステップ

---

## リード：なぜこのnoteは1,500円売れたのか（3回失敗の告白）

結論から言います。このnote、公開から48時間で1,500円売れました。3本。たった3本。でも僕にとっては、人生で初めて自分の書いた文章で得た、本物の売上です。

正直なところ、ここに至るまでに僕は3回失敗しています。「ChatGPTに"副業で稼げる記事書いて"って投げれば一発だろ」と思って全部AI任せにした1本目。「じゃあ丁寧にプロンプト書けばいいんだろ」と頑張ったのに、読み返すと自分でも薄いと感じた2本目。「今度こそは」と気合いで5,000字書いたのに、公開してもXで誰にも反応されなかった3本目。全部、0円でした。

4本目でようやく、売れた。何が違ったのか。才能ではありません。文章力でもない。フォロワー数でもない（僕のXは今でも80人ちょっと）。**違ったのは"AIの使い方"と"公開後の動き方"の2点だけ**です。

このnoteは、その2点をStep1〜7の手順にして、実際に使ったプロンプトを全文載せました。今あなたが読んでいるこの文章自体が、その手順で作られています。つまり、このnoteが売れている時点で、手法が機能していることの証明になっています。

副業を始めたいけど何から手をつければいいか分からない、ChatGPTは触ってるのにお金に繋がらない——そう感じている会社員のあなたに、今日から動ける手順を渡します。

## 第1章：AIに"丸投げ"して3回失敗した話（僕が踏んだ3つの地雷）

最初に言っておきたいのは、僕は文章のプロじゃありません。普通のIT企業の事務系です。コードは書けない。ライター経験もない。ただ、ChatGPTは毎日触っていました。だから「AIで稼げる」って話を聞いた時、正直、余裕だと思ったんですよね。

結果、3回コケました。

**1回目の地雷：AIに全部丸投げした**

「副業で月3万稼げるnote記事を書いて」——これが最初に投げたプロンプトです。出てきた記事、今読み返すと、どこかで見たような内容が3,000字並んでいるだけでした。具体例はなく、数字もなく、誰の悩みにも刺さっていない。当然、0円。

**2回目の地雷：プロンプトを丁寧にしただけ**

反省して、次は「30代会社員向けに、副業で月3万稼ぐ方法を、具体例3つ入れて5,000字で書いて」と指示を細かくしました。出てきた文章は、1回目より少しマシでしたが、今度は別の問題が。**"AIが書いた感"が消えない**。文末が「〜でしょう」「〜と言えます」「〜が重要です」の無限ループ。読者は敏感です。AIっぽさを感じた瞬間にブラウザを閉じます。これも0円。

**3回目の地雷：気合いで書いたが公開後に動かなかった**

3本目は、AI下書きに自分で赤入れして、5,000字を3日かけて仕上げました。内容は悪くなかったと思います。でも、公開してXに1回投稿して、終わり。フォロワー80人の投稿なんて、誰も見ません。公開から1週間、売上0円。

この3つの失敗、実は全部「**やってしまいがちな典型パターン**」です。あなたがこれから踏む予定の地雷と、たぶん同じ。次の章から、どう避けるかを話します。

## 第2章：売れるnoteに必要な3要素（才能ではなく"設計"で決まる）

3回コケて分かったことがあります。売れるnoteと売れないnoteの差は、文章力でも才能でもなく、**3つの設計ポイントを外しているかどうか**だけでした。

**要素1：ジャンル×ペルソナの交差点**

「副業」だけだと広すぎる。「AI副業」でもまだ広い。「非エンジニアの会社員がAIで最初の1円を取る」まで絞って、やっと刺さる。ジャンルは狭ければ狭いほど、その悩みを持つ人に届きます。

**要素2：無料部分で"自分事化"させる**

noteは無料部分だけで読者が「これ、自分のことだ」と感じないと、購入ボタンは押されません。データや知識を無料部分で出しても響かない。響くのは**共感と失敗談**。「僕も同じで悩んでました」が最強の無料コンテンツです。

**要素3：有料部分に"実弾"を入れる**

精神論や抽象論だけだと「500円返して」になります。有料部分にはコピペで使えるプロンプト、実データのスクショ、真似できるテンプレ——つまり**明日使える具体的な武器**を入れる。

この3点を押さえると、文章が多少ぎこちなくても売れます。逆に言うと、文章がどれだけ綺麗でも、この3点が抜けていると売れません。僕の3回の失敗は、全部この3点のどこかが抜けていた、それだけの話でした。

---

**【ここまで無料部分】**

ここから先は、実際に僕が1,500円売れるまでに使った**全プロンプト・無料/有料の切れ目の判定ルール・公開初日のX投稿テンプレ3本**を、一切伏せずに公開します。

具体的には：
- ジャンル選びとペルソナ設計をAIに任せる完全プロンプト
- タイトル・構成・戦略を3連打で出させる方法
- AI臭を消す10プロンプト（僕が実際に使ったもの）
- 無料/有料の切れ目を30〜40%に置く3つの判定ルール
- 公開日に出すX投稿3パターン（共感型・数字型・限定型）の全文テンプレ
- 公開→初売上までの48時間タイムライン（スクショ込み）
- 1本目が売れたら2本目を書くためのテンプレ化手順

500円で、全部渡します。コピペすればそのまま使えます。

## 第3章：【Step1-2】ジャンル選びとペルソナ設計の全プロンプト

ここからが本題です。最初にハッキリ言っておくと、ジャンル選びとペルソナ設計は、**AIに任せるべき工程**です。多くの人が"自分の頭で考えよう"としてここで消耗して、そのままnoteを書き始める前に力尽きます。僕もそうでした。

**ジャンル選定プロンプト（コピペOK）**

```
私は副業でnoteを売って「最初の1円」を取りたい会社員です。
以下の方向性で、500円noteで売れそうなジャンルを10個提案してください。

方向性: {あなたの方向性 例:AI副業、節約、転職}

各ジャンルに以下を添えてください:
- なぜ売れるか（検索需要・競合状況・リピート性の3観点で1〜2行）
- 推奨度★1〜★5
- "最初の1円"までの最短ルート（どの媒体でどう売るか）
```

コツは、10案出させること。1案だと比較ができない。10案あれば、自分の中で「これは絶対違う」「これは意外とアリ」が見えます。僕は1案ずつ出させて3日迷った挙句、結局10案出させ直しました。最初から10案でいいです。

**ペルソナ設計プロンプト（コピペOK）**

```
確定ジャンル「{選んだジャンル}」の500円note購入者として、
具体的なペルソナを3パターン作ってください。

各ペルソナに以下を含めてください:
- 年齢・性別・職業・年収/手取り
- 現在の悩み（セリフ形式で3つ）
- 求めている情報の形式（ステップ手順型 / 体験談型 / データ型）
- 購入を決める瞬間のトリガー（「この文を読んだら買う」の具体文）
```

ここでの裏技は、**"悩みをセリフ形式で"と指定する**こと。普通に「悩み3つ」と頼むと、AIは「副業に関する不安」みたいな抽象語を返してきます。セリフ形式を強制すると「1円でいいから稼いだ実感がほしい」という、そのまま本文のリードに使える言葉が出てきます。

ペルソナは3案出させて1つ選ぶ。選ぶ基準は"自分が昔いた場所"に一番近いもの。未来の読者より、過去の自分の方が解像度高く書けるからです。

## 第4章：【Step3-4】タイトル10案→構成→戦略をAIに作らせる

ジャンルとペルソナが決まったら、あとは3連打で一気に骨組みを作ります。

**タイトルプロンプト（10案出させる）**

```
ジャンル「{確定ジャンル}」をペルソナ「{ペルソナの1行サマリ}」向けに売る
500円noteのタイトルを10個考えてください。

条件:
- 数字・実績・具体性の3要素のうち最低2つを含める
- 各タイトルに使われている型を明記
  A: 金額提示 / B: ステップ数 / C: 警告 / D: 実績
- 思わずクリックしたくなる長さ（30〜50文字目安）
```

僕が採用したのは型Dの実績型でした。実を言うと、最初は型Aの金額提示型（「500円で始める〇〇」）に惹かれたんですが、ペルソナが「数字に釣られる警戒心の強い層」だったので、**実績＋失敗談のセット**が一番刺さると判断しました。

**構成プロンプト（章立て）**

```
タイトル「{確定タイトル}」のnote記事の構成を作ってください。
{ペルソナ}が最後まで読みたくなる章立てにしてください。

各章に以下を記載:
- 章タイトル
- その章で読者に伝えたい1メッセージ
- 読者の感情の動き（共感→希望→確信→行動のどこにあたるか）
- 目安文字数
- 無料/有料のどちらに配置すべきか

章数は7〜10章を目安に。
```

**感情の流れ指定**がミソです。AIは指定しないと全部同じトーンで書いてきます。「共感→希望→確信→行動」と指定すると、章ごとのトーンが自然に変わります。

**戦略プロンプト**

```
{ペルソナ}向け{ジャンル}noteのコンテンツ戦略を作ってください:

1. 読者の悩み（セリフ形式で5個）
2. 各悩みに対して記事で解決すること（1対1で対応）
3. 読後の行動（「今日中に〇〇」「48時間以内に〇〇」の期限付き具体行動）
```

人間がやるのは**選ぶだけ**。ジャンル10案から1つ、ペルソナ3案から1つ、タイトル10案から1つ、構成から章を絞る。全部で10〜15分で済みます。

## 第5章：【Step5】AI臭を消す本文生成（僕が使った10プロンプト）

ここが3回目の失敗から一番学んだところです。AI出力そのままは、絶対に売れません。読者は「AIが書いた文章」を秒で見抜きます。でも、全部自分で書き直すのは無理です。**AI下書き＋10分の人間編集**が黄金比です。

**章ごとの本文生成プロンプト**

```
以下の構成の「第{N}章: {章タイトル}」を書いてください。

前提:
- 読者: {ペルソナ1行サマリ}
- 1メッセージ: {その章で伝えたいこと}
- 目安文字数: {指定文字数}

条件:
- 専門用語を使わず、具体例を入れる
- 章の最後に次の章への橋渡しを1行入れる
- 「結論として」「さらに」「重要です」を使わない
- 文末の「〜でしょう」「〜と言えます」を避ける
```

**禁止語の指定**が効きます。「結論として」「さらに」「重要です」はAIが好んで使う"AI臭"の三大語です。これを禁止するだけで、出力が一段人間寄りになります。

**AI臭を消す10の操作（本文完成後に適用）**

1. **文末の重複を避ける**：同じ語尾が3回連続しないように語尾を混ぜる
2. **感情を言葉にする**：「正直ビビった」「思わず笑った」等、書き手の感情を動かす
3. **読者への問いかけを2箇所**：「〇〇って思いませんか？」を要所に入れる
4. **構成を少し崩す**：教科書的な順序ではなく「思い出したんですが」で脱線を1回
5. **禁止接続詞**：「結論として」「さらに」「重要です」を全削除
6. **失敗体験を1つ挿入**：数字入りの具体的な失敗談
7. **語尾に揺らぎ**：「です・ます」ベースに時々「だ・である」や体言止め
8. **主観を強く出す**：「僕はこう思う」「これが一番効く」と言い切る
9. **フィラーを入れる**：「正直なところ」「実を言うと」を自然に混ぜる
10. **文の長さを不揃い**：短文と長文を交互に

10個全部やる必要はなくて、**最低3つ**で十分効きます。僕は①⑥⑨の3つを毎回やってます。この3つで、AI臭の80%は消えます。

実際に今あなたが読んでいるこの本文も、この3つを適用した後のものです。完璧ではないけど「人間が書いた感」は出てると思いませんか？

## 第6章：【Step6】無料/有料の切れ目を決めた実データ

無料部分の切れ目、どこで切るか迷いますよね。僕は3回失敗した時、全部ここを感覚で決めていました。今は違います。**3つの判定ルール**で決めます。

**ルール1：全体の30〜40%地点**

これはnote公式も推奨している黄金比です。20%だと無料で引き込む前に切れる。50%を超えると「もう十分読んだ」と満足されて買われない。30〜40%が、続きが気になる手前のギリギリ地点。

**ルール2：切れ目の直前に"予告文"を入れる**

「この先で、実際に使ったプロンプトを全文公開します」——この1文があるかないかで、購入率は体感2倍違います。予告文のテンプレ：

```
ここから先は、実際に僕が{実績}するまでに使った
{具体的な成果物1}・{具体的な成果物2}・{具体的な成果物3}を、
一切伏せずに公開します。
```

**ルール3：共感の山で切る（知識の山では切らない）**

切れ目直前に置くべきは、読者が「この人、自分と同じだ」と最も強く感じる瞬間です。データや手順の途中で切ると「知りたいな」程度で終わります。共感の山で切ると「続きを読まずにいられない」になります。

僕のこのnoteでは、第2章の「3回の失敗は、全部この3点のどこかが抜けていた」の直後で切りました。読者が「じゃあその3点をどう押さえるんだ」と最も知りたくなる瞬間です。

## 第7章：【Step7】公開日に売るためのX投稿テンプレ3本

ここを飛ばすと、どれだけ良いnoteを書いても売れません。3回目の失敗の原因、実はこれでした。**noteは"公開して終わり"ではなく、"公開日から始まる"** です。

公開日に出すべき投稿は3パターン。朝・昼・夜で1回ずつ、計3回投稿します。

**パターンA：共感型（朝8時）**

```
副業したいけど何から始めればいいか分からない、
ChatGPTは触ってるのに稼ぎに繋がらない——
1ヶ月前の僕がそうでした。

3回失敗した末に4本目で初売上1,500円。
何が違ったのか、全部noteにまとめました。
[noteリンク]
```

**パターンB：数字型（昼12時）**

```
公開48時間で1,500円売れました。
フォロワー80人、文章経験ゼロの会社員が、
AIで500円noteを作って"最初の1円"を取った全手順を
7ステップで公開しています。

プロンプトも全部載せました。
[noteリンク]
```

**パターンC：限定型（夜21時）**

```
AIにnote書かせて全然売れなかった人向けに、
実際に使ったプロンプトと無料/有料の切れ目の作り方を
500円で全部公開しました。

公開記念で今週は値上げしません。来週から700円予定。
[noteリンク]
```

3投稿の鉄則：**絶対にリンクを3回全部貼る**。1回だけ貼って残りは"プロフから"は弱い。流れてきた人がワンクリックで飛べるようにする。

## 第9章：次の1本を書くためのテンプレ化手順

最後に、ここが実は一番大事です。1本売れたら、2本目で稼ぐ。

このnoteで使った10章構成、そのまま別ジャンルに横展開できます。テンプレ化の手順は3ステップ。

**Step1：ジャンルだけ差し替える**
「AIで500円noteを作る」→「AIで動画編集の副業を始める」等。構成の骨格は同じ。

**Step2：ペルソナだけ作り直す**
新ジャンルの購入者を第3章のプロンプトで3案出させて選ぶ。ここは手を抜かない。

**Step3：実績の数字を更新する**
1本目で1,500円売れたなら、2本目のリードには「1本目は48時間で1,500円、今回は〜」と繋げる。**実績の積み上げが次の購入理由になる**。

2本目以降は、1本目の30%の時間で作れます。なぜなら、構成もプロンプトも、使い回せるから。

ここまで読んでくれたあなたなら、もう分かってると思います。**最初の1円は、センスでも才能でもなく、手順**です。今日、note開設からやってみてください。1週間後、あなたも僕と同じダッシュボードを見ているはずです。

一緒にやりましょう。
