# ジャンル確定

## 確定ジャンル
**AIを使って500円noteを作り、売って「最初の1円」を得る方法**

## 短縮名
`ai-note-making`

## なぜこのジャンルか
- 検索需要: 「note 稼ぐ」「AI 副業 note」など、noteで収益化したい層が常に存在
- 競合状況: "AI×note制作"に絞った実践型ノウハウはまだ飽和していない
- リピート性: 1本目が売れたら2本目・3本目とテンプレ横展開できるため継続購入/リピート層を狙える
- メタ構造の強み: このnote自体がその手順で作られた成果物になる=最強の実績証明

## 想定ポジション
- 読者: 副業したいが何から始めるか分からない会社員、noteを書きたいが売り方が分からない人
- 主張: AIをうまく使えば、ライティング経験ゼロでも500円noteで"最初の1円"は取れる
- 差別化: 抽象論ではなく「使ったプロンプト・無料/有料の切れ目・X投稿文」まで全部出す実弾型

## 売れる3基準チェック
- [x] 悩んでいる人が多い（副業×AI×note は三大検索キーワードの交差点）
- [x] 競合が多すぎない（"AI noteの作り方"実践型はまだ空きがある）
- [x] 調べていてストレスを感じない（書き手が実際に作った経験で語れる）
