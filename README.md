# AInote

500円の有料 [note](https://note.com/) 記事を、AI（Claude Code 等）と一緒に企画・執筆・公開・集客まで進めるための作業用リポジトリです。

## 含まれるもの

| 内容 | 説明 |
|------|------|
| **`.claude/skills/note-500yen/`** | 8ステップ（ジャンル → ターゲット → タイトル/構成 → 戦略 → 本文 → note設定 → X集客 → 改善）で対話しながら記事化する [Agent Skill](https://code.claude.com/docs/en/agents-and-tools/agent-skills/overview)。各ステップで承認を取りつつ、売れる構成のガイドと中間成果物の保存方針が定義されています。 |
| **`drafts/`** | 執筆中の下書きとメモ（`genre.md` / `persona.md` / `body.md` など、ステップごとの Markdown）。 |

## 使い方

1. このリポジトリをローカルに clone する。
2. Claude Code（または同スキルに対応する環境）で `note-500yen` スキルを有効にする。
3. スキルに沿って対話。成果物の保存先はスキル内のパス定義に合わせ、必要なら自環境用にパスを書き換える。

初回向けの目安（スキル内案内）として、4週間で Step①〜⑧を進める想定も用意されています。一気に進めても、週単位で区切っても構いません。
